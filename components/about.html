01. Tentang Saya

Saya adalah lulusan Sarjana Terapan Teknik Telekomunikasi dari Politeknik Elektronika Negeri Surabaya dengan IPK 3.58. Sebelumnya, saya menempuh pendidikan di SMKN 1 Jenangan Ponorogo mengambil jurusan Rekayasa Perangkat Lunak.

Fokus saya adalah menggabungkan solusi perangkat lunak dengan perangkat keras. Saya memiliki ketertarikan besar dalam riset dan implementasi sistem cerdas, baik melalui pembuatan aplikasi web dinamis maupun pengembangan sistem kontrol robotik menggunakan protokol IoT.
